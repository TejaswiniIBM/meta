<h1 align="center">ZOPEN-IMPORTENVS</h1>

<h2>NAME
<a name="NAME"></a>
</h2>

<p style="margin-left:11%; margin-top: 1em">zopen-importenvs
&minus; manual page for zopen-importenvs 0.8.0</p>

<h2>DESCRIPTION
<a name="DESCRIPTION"></a>
</h2>

<p style="margin-left:11%; margin-top: 1em">USAGE: .
./zopen&minus;importenvs [path to buildenv to fetch
dependency] If no argument is specified,
zopen&minus;importenvs will source all .env files from
<i>/var/lib/jenkins/zopen/prod</i> and
<i>/var/lib/jenkins/zopen/boot</i> If an argument is
specified and it represents a buildenv file,
zopen&minus;importenvs will source the ZOPEN_*_DEPS
specified dependencies from
<i>/var/lib/jenkins/zopen/prod</i> and
<i>/var/lib/jenkins/zopen/boot</i></p>
